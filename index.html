Codenotes

A Text Editor for Computer Science Students

View on GitHub

CodeNotes is a editor I built in order to learn and practice my JavaScript skills.

In school, I found myself struggling to find the best way to take notes for my computer programming course. I found myself needing functionality for code formatting but also needing simple functions such as bolding and bullet points.

With that, the CodeNotes text editor was born.

Its purpose is to help computer science students notes by providing a code editor and a rich - text editor side by side.

The text area utilizes the behave.js library to have code formatting capabilites. Local storage is used to store both code and text.
